# 📁 Assets — Lembrança Cantada

Suba aqui os vídeos e imagens do site. Depois é só me avisar que eu troco cada
placeholder (caixa tracejada) pelo arquivo certo no lugar certo.

Tudo nesta pasta fica acessível no site pelo caminho `/assets/...`
(ex.: `public/assets/hero/mockup.mp4` → `https://musicas.linkarbox.app/assets/hero/mockup.mp4`).

## Onde cada arquivo entra

| Pasta | Placeholder no site | Sugestão de arquivo | Formato ideal |
|-------|--------------------|--------------------|---------------|
| `hero/` | Hero (lado direito) | `mockup.mp4` ou `mockup.png` | vídeo vertical 9:16 ou print do app |
| `passos/` | Como Funciona (4 passos) | `passo-1.png` … `passo-4.png` | imagem 4:3, fundo claro |
| `previa/` | Prévia / carrossel | `previa.mp4` | vídeo vertical 9:16 |
| `recursos/` | Bento "Voz à sua escolha" | `voz.png` | mockup 9:16 |
| `recursos/` | Bento "Vídeo de brinde" | `video-brinde.mp4` | vídeo 16:9 |
| `demo/` | Banner Demo | `demo-1.mp4` / `demo-2.mp4` | vídeos verticais |
| `og/` | Compartilhamento (link preview) | `og-image.png` | 1200×630 |

## Dicas
- **Vídeos:** MP4 (H.264), de preferência < 8 MB cada (carrega rápido). Pode ser sem áudio.
- **Imagens:** PNG ou WEBP. Nomeie em minúsculo, sem espaço (use `-`).
- Não precisa ser perfeito agora — sobe o que tiver que eu encaixo e ajusto o enquadramento.
